1. 파일을 다운로드하고 압축을 해제한다
2. chrome://extensions 에 들어간다
3. 개발자 모드를 킨다
4. "압축해제된 확장 프로그램을 로드합니다." 를 누르고 다운로드한 폴더 (압축 해제된) 선택
5. 사이트에 들어가면 모든 이미지가 바뀜
6. chrome://extensions 에서 on / off 가능
다운받은 폴더를 지우거나 위치를 바꾸면 확장 프로그램이 작동 안할수도 있음
